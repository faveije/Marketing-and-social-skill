Social Media Generator
A powerful skill that automatically generates optimized social media posts for Twitter/X, LinkedIn, Instagram, and Telegram from any topic or announcement.

Features
🎯 Platform-Optimized: Each post is tailored to the specific platform's best practices
📊 Character-Aware: Respects character limits and recommends optimal lengths
🏷️ Smart Hashtags: Generates relevant hashtags based on the topic
💡 Engagement Tips: Provides suggestions to maximize post engagement
🎨 Media Suggestions: Recommends appropriate media types for each post
How It Works
1. Input Processing
The skill analyzes your topic/announcement to identify:

Key themes and keywords
Target audience
Tone and sentiment
Relevant industries or sectors
2. Platform Adaptation
Based on the selected platform, it adjusts:

Length: Optimizes character count
Tone: Matches platform culture (professional for LinkedIn, casual for Twitter, etc.)
Hashtags: Applies platform-specific hashtag strategies
Formatting: Uses appropriate emojis, line breaks, and structure
3. Post Generation
Creates multiple post variations with:

Hook/headline
Main content
Call-to-action
Hashtags
Engagement suggestions

Python API
Copyfrom generate_posts import generate_post

# Generate a Twitter post

)

Parameters
Parameter	Type	Required	Description
topic	string	Yes	The subject or announcement to convert
platform	string	Yes	One of: twitter, linkedin, instagram, telegram
Output
The function returns a dictionary with:

content: The generated post text
hashtags: List of suggested hashtags
media_suggestions: Recommended media types
character_count: Length of the post
engagement_hooks: Tips to boost engagement
Platform Guidelines
Twitter/X
Limit: 280 characters
Tone: Punchy, conversational
Hashtags: 1-3 max
Best for: Quick updates, news, engagement
LinkedIn
Limit: 3000 characters (optimal: 150-300)
Tone: Professional, informative
Hashtags: 3-5 industry-relevant
Best for: Business announcements, thought leadership
Instagram
Limit: 2200 characters (optimal: 100-150)
Tone: Visual, authentic, inspiring
Hashtags: 5-15 relevant
Best for: Visual content, behind-the-scenes, community
Telegram
Limit: 4096 characters
Tone: Direct, informative, community-focused
Hashtags: 2-4 or none
Best for: Updates, links, community announcements

Social Media Generator
A powerful skill that automatically generates platform-optimized social media posts from any topic or announcement. Perfect for creators, marketers, and businesses looking to maintain consistent social media presence across multiple platforms.

Features
✨ Multi-Platform Support: Twitter/X, LinkedIn, Instagram, and Telegram
📝 Platform-Aware Formatting: Each post optimized for specific platform constraints
🎯 Smart Hashtag Generation: Relevant hashtags tailored to each platform
🎨 Tone Adaptation: Different tones for different platforms (professional for LinkedIn, casual for Twitter, etc.)
📊 Character Count Validation: Ensures posts meet platform limits
🚀 One-Command Generation: Generate posts for single or all platforms instantly
How It Works
The skill uses intelligent text generation to:

Analyze your topic to understand the core message
Adapt tone and style based on the target platform
Optimize length to fit platform character limits
Generate relevant hashtags for discoverability
Add engagement elements like questions or calls-to-action
Platform Guidelines
Twitter/X
Maximum 280 characters
1-3 hashtags
Conversational, punchy tone
Hook readers in first 100 characters
LinkedIn
Up to 3,000 characters
3-5 professional hashtags
Professional, value-driven tone
Include clear call-to-action
Instagram
Up to 2,200 characters
5-15 hashtags
Visual, authentic tone with emojis
Storytelling format
Telegram
Up to 4,096 characters
2-5 hashtags
Direct, informative tone
Markdown formatting support
